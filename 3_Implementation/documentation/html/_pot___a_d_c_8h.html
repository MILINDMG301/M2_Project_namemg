Servo_motor_control_using_Potentiometer
Potentiometer value is connected to ADC pin of the atmega328 microcontroller. The ADC pin will enable ADC and convert the signal into digital. Which is then converted into PWM signal to rotate the servo motor by specific angle according to the input value of the potentiometer.
